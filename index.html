Installation automatique de cartes

Pour installer les cartes, cliquez sur le bouton depuis votre téléphone (il faut que Locus Map ou Cartograph Map soit déjà installé sur le téléphone).

Cartes de base

  • France - IGN : scan topo, plan, orthophoto, pente, iti skirando
  • Suisse - Swisstopo : scan topo, pente, iti ski, iti VTT, iti pédestre, arrêt TP
  • Espagne : scan topo, orthophoto
  • Autriche : scan topo
  • Monde - Google : plan, satellite, vélo
  • Monde - WayMarkedTrail OSM : surcouche itinéraire pédestre, vélo, VTT, cheval, ski piste, roller
  • Monde - Bing : ortho
